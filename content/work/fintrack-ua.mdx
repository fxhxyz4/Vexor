## Про проєкт

FinTrack UA — фінтех-стартап, який допомагає малому бізнесу відстежувати фінанси в реальному часі. До нас звернулись з задачею: зробити лендинг, який конвертує відвідувачів у реєстрації.

## Задача

Клієнт мав MVP продукту але не мав сайту. Потрібно було:

- Пояснити цінність продукту просто і зрозуміло
- Показати інтерфейс через живу демонстрацію
- Підключити оплату підписки через Stripe
- Зробити SEO під UA ринок

## Рішення

Ми розробили лендинг на Next.js з анімованим превью дашборду прямо на сторінці. Замість скріншотів — живі CSS-анімації, які імітують роботу з продуктом.

Ключові рішення:

- **Анімований дашборд** — замість відео, 60fps CSS анімація
- **Stripe Checkout** — інтеграція підписки без переходу на сторонні сторінки
- **Core Web Vitals** — LCP < 1.2s, CLS = 0

## Результат

- Конверсія лендингу: **4.2%** (середня по ринку 1.8%)
- Завантаження сторінки: **< 1.5s** на мобільних
- **300+** реєстрацій за перший місяць
