## Про проєкт

DeliveryPro — сервіс кур'єрської доставки в Києві. Клієнт звернувся з потребою в мобільному застосунку для кур'єрів і окремому додатку для диспетчерів.

## Задача

- Застосунок для кур'єрів: маршрути, статуси замовлень, навігація
- Панель диспетчера: розподіл замовлень, відстеження в реальному часі
- Push-сповіщення при зміні статусу замовлення
- Офлайн-режим для нестабільного з'єднання

## Рішення

Розробили два застосунки на React Native — для iOS та Android одночасно. Інтегрували Google Maps для маршрутизації та Firebase для real-time оновлень статусів.

Ключові рішення:

- **Офлайн-перший підхід** — дані кешуються локально, синхронізуються при появі з'єднання
- **Background location** — трекінг кур'єра навіть при згорнутому додатку
- **Push через FCM** — миттєві сповіщення клієнтам і диспетчерам
