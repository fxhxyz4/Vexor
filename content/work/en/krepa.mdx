## About the project

Krepa is a mobile workout app. The team needed a promotional landing page that would convince users to download the app and showcase its features.

## Challenge

- Bold, energetic design matching the fitness brand
- Animated sections demonstrating app features
- CTA buttons for App Store and Google Play
- Responsive across all devices
- Fast load times

## Solution

We built the landing page on Next.js with Framer Motion animations. Each section animates on scroll, showcasing app features through smooth transitions.

Key decisions:

- **Scroll animations** — sections animate as they enter the viewport
- **Gradient design** — brand colors via CSS variables
- **App Store / Google Play** — deeplinks for direct store navigation
- **Core Web Vitals** — optimized for fast loading

## Results

- Landing conversion: **6.1%** to download
- Lighthouse score: **98/100**
- Responsive from 320px to 2560px
