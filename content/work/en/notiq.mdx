## About the project

Notiq is a cross-platform note-taking app. The goal was to create a powerful yet simple tool for Desktop, iOS, Android and Web — fast, clean, and intuitive.

## What we built

- Unified design system across all platforms
- Minimal interface with no distractions
- Adaptive grid — one design scales from mobile to desktop
- Dark and light themes
- Interactive prototype in Figma

## Results

A reliable, distraction-free app that adapts to different devices. Notiq helps users capture thoughts, organize knowledge, and stay productive anywhere.
