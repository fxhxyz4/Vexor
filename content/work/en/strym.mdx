## About the project

Strym is a full-featured e-commerce app for electronics. Dark editorial style, bilingual UI (UA/EN), theme switcher and built-in admin panel.

## What we built

- Catalog with full-text search, filtering by category, brand and price
- Product pages with photo gallery, specs and reviews
- Cart — guest mode via localStorage and authenticated via DB with sync on login
- Checkout with two delivery options and order status tracking
- JWT auth with refresh tokens and httpOnly cookies
- Admin panel: full CRUD for products, categories, orders and users

## Results

- Catalog load time: **< 1s** thanks to SSR + ISR
- Swagger UI with auto-generated API docs
- Deploy: Frontend on Vercel, Backend on Railway, PostgreSQL as Railway plugin
