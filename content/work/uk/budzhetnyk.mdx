## Про проєкт

Retail Group UA — мережа онлайн-магазинів. Клієнт використовував три різні системи для управління замовленнями, складом і клієнтами. Потрібна була єдина платформа.

## Задача

- Об'єднати замовлення з трьох магазинів в одну панель
- Аналітика продажів з фільтрацією по датах, категоріях, менеджерах
- Управління складськими залишками з алертами
- Ролі та права доступу для команди з 15 людей

## Рішення

Розробили SaaS-платформу на Next.js App Router з PostgreSQL через Prisma. Вся бізнес-логіка через typed API routes, інтерфейс з Tailwind + Radix UI.

Ключові рішення:

- **Real-time дашборд** — WebSocket для живого оновлення статусів
- **RBAC** — гнучка система ролей: адмін, менеджер, склад, аналітик
- **Bulk operations** — масові дії з замовленнями через чергу задач
- **Export** — звіти в Excel та PDF за будь-який період

## Результат

- **3x** швидше обробка замовлень порівняно з попередньою системою
- **0 помилок** при розподілі складських залишків за перші 3 місяці
- Команда з 15 людей повністю перейшла за **2 тижні**
