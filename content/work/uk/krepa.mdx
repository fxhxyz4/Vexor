## Про проєкт

Krepa — мобільний застосунок для тренувань. Команда потребувала промо-лендинг який переконає користувачів завантажити застосунок та розкаже про його переваги.

## Задача

- Яскравий, енергійний дизайн в стилі фітнес-бренду
- Анімовані секції з демонстрацією функцій застосунку
- CTA кнопки для App Store та Google Play
- Адаптивність для всіх пристроїв
- Висока швидкість завантаження

## Рішення

Розробили лендинг на Next.js з Framer Motion анімаціями. Кожна секція з'являється при скролі, демонструючи функції застосунку через плавні переходи.

Ключові рішення:

- **Scroll animations** — секції анімуються при появі у viewport
- **Градієнтний дизайн** — фірмові кольори бренду через CSS змінні
- **App Store / Google Play** — deeplinks для прямого переходу в магазини
- **Core Web Vitals** — оптимізовано для швидкого завантаження

## Результат

- Конверсія лендингу: **6.1%** переходів до завантаження
- Lighthouse score: **98/100**
- Адаптивний від 320px до 2560px
